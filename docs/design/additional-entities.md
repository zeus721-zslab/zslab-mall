# 추가 추천 엔티티 (상업용 대형몰 기준)
 
핵심 도메인 외 상업 운영을 위해 필요한 엔티티 목록.
 
## 추천 엔티티
 
| 엔티티 | 용도 | 우선순위 |
|--------|------|:--------:|
| Coupon | 쿠폰 발급·사용 관리 | 중 |
| Promotion | 프로모션·할인 이벤트 | 중 |
| Inventory | 재고 관리 (Product.stock 분리) | **상** |
| Cart | 장바구니 | **상** |
| Review | 상품 리뷰·평점 | 중 |
| Point | 적립금·포인트 | 중 |
| Payment | 결제 정보·PG 연동 | **상** |
| Delivery | 배송 추적·운송장 | **상** |
| Claim | 클레임 (취소/반품/교환 요청) | **상** |
| Refund | 환불 처리 | **상** |
| Notification | 알림 (이메일/SMS/푸시) | 중 |
| AuditLog | 감사 로그 (운영자 행위 추적) | **상** |
| Banner | 배너·메인 노출 관리 | 하 |
| Category | 상품 카테고리 (계층 구조) | **상** |
| SearchIndex | 검색 인덱스 (Elasticsearch 연동) | 중 |
| Warehouse | 창고·물류 거점 | 하 |
 
## 상업용 필수 3종
 
- **AuditLog** — 운영자 행위 추적 (권한 변경·정산 처리·상품 강제 수정 등). 분쟁·감사 대응 필수.
- **Settlement 정합성을 위한 Inventory** — `Product.stock` 단일 필드로는 동시성·이력 추적 한계.
- **Claim / Refund** — 결제 분쟁 처리 흐름. PG·통신판매업 규제 대응.
